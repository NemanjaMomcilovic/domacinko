# Domaćinko

AI-pokretan pomoćnik za domaćinstvo — prvi proizvod platforme **10KEY**.

## Pokretanje

Otvorite `index.html` u browseru ili direktno `pages/home.html`:

```
C:\Users\pc\10KEY\domacinko\index.html
```

Za lokalni server (opciono):

```powershell
cd C:\Users\pc\10KEY\domacinko
python -m http.server 8080
```

Zatim otvorite: http://localhost:8080

## Struktura

- `pages/` — HTML stranice
- `css/` — Dizajn sistem
- `js/` — Logika i localStorage
- `docs/` — Dokumentacija

## Tehnologije

Čist HTML, CSS i JavaScript — bez frameworka. Podaci se čuvaju u `localStorage`.
