Politika privatnosti

Poslednje ažuriranje: jul 2026 · Domaćinko v7.0.3

1. Ko smo mi

Domaćinko je aplikacija za upravljanje domaćinstvom, deo platforme 10KEY. Ovu politiku vodi tim iza Domaćinka / 10KEY projekta.

2. Koje podatke prikupljamo

  • Email i nalog — ako se registrujete (Supabase autentifikacija)
  • Profil — ime, finansijski ciljevi, valuta (opciono)
  • Podaci domaćinstva — troškovi, kupovina, članovi porodice, održavanje i slično
  • Beta feedback — ocena, komentari, opcioni kontakt email
  • localStorage — podaci na vašem uređaju dok ne prijavite nalog
  • OpenAI API ključ — samo ako ga sami unesete u Podešavanjima (čuva se lokalno na uređaju)

3. Gde se podaci čuvaju

Ako koristite nalog, podaci se sinhronizuju preko Supabase (hosting u EU regionu, npr. Frankfurt). Supabase primenjuje Row Level Security — svaki korisnik vidi samo svoje podatke (ili deljeno domaćinstvo ako ste se pridružili porodici).

U gost režimu podaci ostaju isključivo u pregledaču na vašem uređaju (localStorage).

4. Kako koristimo podatke

  • Pružanje funkcija aplikacije (budžet, sinhronizacija, AI saveti)
  • Poboljšanje proizvoda na osnovu beta feedbacka
  • Tehnička podrška ako nas kontaktirate

Ne prodajemo vaše podatke trećim stranama.

5. Treće strane

  • Supabase — autentifikacija i baza podataka
  • Google / Facebook — samo ako izaberete prijavu preko društvenih mreža
  • OpenAI — samo ako unesete svoj API ključ za napredni AI
  • GitHub Pages — hosting statičkih fajlova aplikacije

6. Vaša prava

Možete izvesti podatke (Podešavanja → Izvezi JSON), obrisati nalog (Profil → Obriši nalog) ili resetovati lokalne podatke. Za zahteve u vezi privatnosti pišite na kontakt ispod.

7. Bezbednost

Koristimo HTTPS, RLS u bazi i javni (anon) Supabase ključ u frontendu — nikad service_role tajnu. Vi ste odgovorni za čuvanje lozinke i uređaja na kojem koristite aplikaciju.

8. Beta period

Tokom beta testiranja aplikacija može menjati funkcije. Feedback koji pošaljete može biti pregledan u Supabase Table Editoru radi poboljšanja proizvoda.
